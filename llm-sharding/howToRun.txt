To enter the virtual environment put this in the console: .\.venv\Scripts\Activate.ps1

How to run: python shard_model.py

Basically explained what I am doing:
    -  I should generate onnx files and safetensor files plus the json config files (the sharded LLM) since the ONNX files
    will help transformers.js run the LLM in the browser and the safetensor files will be used to train the LLM on the data that the
    user provides using LoRA training.

Command to export LLM to onnx format:
    optimum-cli export onnx `
        -m .\shard-models\qwen2.5-1.5b-instruct `
        --task text-generation-with-past `
        --device cpu `
        --opset 17 `
        --batch_size 1 `
        .\shard-models\qwen2.5-1.5b-instruct\onnx
