Run this to setup the proper libraries to train your model:
    - pip3 install transformers datasets accelerate peft pillow
    This download of pytorch allows for GPU support:
        - pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu126

    myenv\Scripts\Activate.ps1

    uvicorn main:app --reload
